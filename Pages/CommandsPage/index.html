Стандартные команды

Команда Хелп

Основная команда, которая поможет получить весь список команд.

Формат ввода:

User

!Хелп

Команда Очистить

Команда, которая поможет Вам очистить сообщения в канале. Есть два условия: не более 100 сообщений за раз, и сообщения не должны быть отправлены позже 2 недель. Ты должен указать количество сообщений на первом месте после названия команды.

Формат ввода:

User

!Очистить 20

Ботя

Начинаю удаление сообщений...

Ботя

Удаление сообщений произведено успешно

Команда Жалоба

Данная команда позволяет отправить Вам жалобу на участника сервера администратору.

Формат ввода:

User

!Жалоба @AnotherUser Нарушил правила сообщества Discord

Ботя

Жалоба на участника отпралена на рассмотрение владельцу сервера.

Команда Кик

Данная команда позволяет кикнуть пользователя с сервера. У Вас должно быть право кикать на этом сервере, чтобы пользоваться этой командой.

Формат ввода:

User

!Кик @AnotherUser

Ботя

Я кикнул AnotherUser.

Команда Бан

Данная команда позволяет забанить пользователя на сервере. У Вас должно быть право банить на этом сервере, чтобы пользоваться этой командой.

Формат ввода:

User

!Бан @AnotherUser

Ботя

Я забанил AnotherUser.

Команда МедленныйРежим

Данная команда позволяет включить медленный режим в канале, где Вы написали эту команду.

Формат ввода:

User

!МедленныйРежим 60

Ботя

В канале Текстовый канал включен медленный режим. Интервал: 60 секунд

Команда Рассылка

Данная команда рассылает всем участникам сервера текст, который Вы написали боту.

Формат ввода:

User

!Рассылка На моем YouTube канале вышло новое видео! Обязательно посмотрите его!

Ботя

Рассылка произведена успешно.

Команда ДобавитьСоединение

Данная команда позволит Вам установить "соединение" с другим сервером. Иначе говоря Вы соединяете два текстовых канала на разных серверах. Для того чтобы установить соединение нужно указать Id канала, который находится на другом сервере.

Формат ввода:

User

!ДобавитьСоединение 0000000000000000

Ботя

Соединение добавлено.

Команда УдалитьСоединение

Данная команда позволит Вам удалить "соединение" с другим сервером.

Формат ввода:

User

!УдалитьСоединение

Ботя

Удаление произведено успешно.

Команда ДобавитьРольЗаРеакцию

Данная команда позволит вам создать сообщение, у которого будут реакции. Если кликнуть на эти реакции, тогда выдается определенная роль.

Формат ввода:

User

Хэй! Хотите роль @Role? Если да, тогда жмите на реакцию ниже!

1

User

!ДобавитьРольЗаРеакцию 0000000000000000 #💬основной @Какая-нибудь роль

Ботя

Добавлено.

Команда УдалитьРольЗаРеакцию

Удаляет реакцию за роль у данного сообщения

Формат ввода:

User

!УдалитьРольЗаРеакцию 0000000000000000

Ботя

Удалено.

Команды магазина ролей

Команда ДобавитьРоль

Добавляет роль на "продажу" другим участникам сервера. Нужно указать цену и упомянуть роль.

Формат ввода:

User

!ДобавитьРоль 100 @Role

Ботя

Роль добавлена на продажу.

Команда УбратьРоль

Снимает роль с "продажи" другим участникам сервера. Нужно упомянуть роль.

Формат ввода:

User

!УбратьРоль @Role

Ботя

Роль @Role снята с продажи.

Команда КупитьРоль

Покупает роль. Нужно упомянуть роль.

Формат ввода:

User

!Купить @Role

Ботя

Роль @Role куплена.

Кастомизация бота

Команда ДобавитьРольПоумолчанию

Добавляет роль, которая будет выдаваться всем. Нужно упомянуть роль.

Формат ввода:

User

!ДобавитьРольПоумолчанию @Role

Ботя

Роль успешно задана. Она будет выдаваться всем пользователям по-умолчанию.
